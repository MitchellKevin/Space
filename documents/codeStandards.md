# Code Standards: Work D.R.Y. (Don't Repeat Yourself)

**Work D.R.Y. (Don't Repeat Yourself)**

**Varialbe naming:** camelCase name structuring and in English.

**Custom Properties:** With colors use the colors based in the root styling.

**Mergen:** When merging there should be a minimum of 2 people.

**Mapping:** For every feature / element on the page a seperate script & style file. That need to be placed in the mapps scripts and styles.

**Commits:** Should all be in English and formated in first a verb and then a noun.

**Animation naming:** Animations name always start with -- at the start, example: --animation.

**JS varible only with let or const NO var**
